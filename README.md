## Project Topic

**Do Crimes Change on Weekends?**

We analyze San Francisco crime data to compare weekday and weekend patterns across selected crime types.  
The main conclusion is that the weekend effect is not primarily an increase in total crime volume, but a shift in **when** crimes occur, especially for assault.

## Main Findings

- Weekend crime totals are broadly similar to weekday totals  
- Assault shows the clearest weekend night shift  
- Theft remains comparatively stable and follows a more routine daily pattern  
- Spatial clustering adds context to the temporal patterns
